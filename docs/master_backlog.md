# خطة تطوير حلقتي الرئيسية

آخر تحديث: 2026-07-12

هذه الوثيقة هي المصدر الرئيسي للمتطلبات. لا تُعد أي ميزة مكتملة لمجرد وجود كود لها أو ذكرها في سجل الإصدارات؛ يلزم اجتياز معايير القبول على المنصة المحددة.

## قواعد التنفيذ

- حماية بيانات الطلاب قبل أي تغيير في المزامنة أو قاعدة البيانات.
- تنفيذ تغييرات قاعدة البيانات عبر migrations قابلة للمراجعة، ومنع أوامر الحذف في بيئة الإنتاج.
- فصل حالة Android عن الويب في التوثيق والاختبار.
- توثيق كل مجموعة في `CHANGELOG.md` و`docs/release_notes.md`.
- إضافة اختبار يمنع رجوع العطل كلما أُصلح عطل متعلق بالبيانات أو الحسابات.
