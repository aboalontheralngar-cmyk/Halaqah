# حلقتي — نظام إدارة الحلقات القرآنية

نظام متكامل لإدارة الحلقات القرآنية: تسجيل الطلاب، الحضور والانضباط، متابعة الحفظ والمراجعة، النقاط والتحفيز، الاختبارات، والتقارير.

## بنية المشروع

```
Halaqah/
├── website/        # تطبيق الويب الرئيسي (Next.js + Supabase + Tailwind)
├── lib/            # تطبيق Flutter للجوال (يعمل محلياً بـ SQLite)
├── android/ ios/   # منصات Flutter
├── assets/         # موارد تطبيق Flutter (قاعدة بيانات الآيات، الأيقونات)
├── tools/          # ملفات بيانات مساعدة
├── shared/         # توثيق مشترك
└── CHANGELOG.md    # سجل التحسينات والتطويرات
```

## المكونات

### تطبيق الويب (`website/`)
الواجهة الرئيسية للمعلمين والمشرفين — Next.js 16 مع Supabase (قاعدة بيانات + مصادقة).

التشغيل محلياً:
```bash
cd website
npm install
npm run dev
```

يتطلب ضبط متغيرات البيئة في `.env.local`:
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`

مخطط قاعدة البيانات: `website/database_schema.sql`

### تطبيق الجوال (`lib/`)
تطبيق Flutter يعمل دون اتصال (SQLite) مع نسخ احتياطي واستعادة، وستضاف مزامنة Supabase لاحقاً.

التشغيل:
```bash
flutter pub get
flutter run
```

## المزايا الرئيسية

- إدارة الطلاب والحلقات والمراكز
- تسجيل الحضور والتأخير والإجازات (مع QR مشفر)
- متابعة الحفظ والمراجعة بدقة الآية والسطر
- نظام نقاط وتحفيز وعقوبات
- اختبارات وتقارير PDF (مع التاريخ الهجري)
- لوحة شرف ومتابعة أولياء الأمور
