## التشغيل والإعداد

### متطلبات النظام
- Node.js 18+
- npm / yarn / pnpm
- حساب Supabase (للإنتاج)

### التشغيل محلياً

**تطبيق الويب:**
```bash
cd website
npm install
npm run dev
```

الويب متاح على `http://localhost:3000`

**تطبيق الجوال:**
```bash
flutter pub get
flutter run
```

### متغيرات البيئة

في `.env.local` للويب:
```
NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
```

### قاعدة البيانات

تطبيق الويب يتطلب Supabase:
1. أنشئ مشروع Supabase جديد
2. طبّق `website/database_schema.sql` على قاعدة البيانات
3. ضع بيانات المصادقة في `.env.local`

ملف التوسعة `website/database_schema_extensions.sql` (329 سطر) يحتوي على:
- جداول إضافية كاملة (صندوق، خطط، تقارير، إلخ)
- سياسات RLS آمنة لكل جدول
- فهارس أداء
- جاهز للتطبيق الفوري
