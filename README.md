# تطبيق الحلقة القرآنية - Halaqah App

## 📝 وصف المشروع
تطبيق مبسط لإدارة الحلقات القرآنية يتكون من:
- تطبيق موبايل للطلاب (Flutter)
- لوحة تحكم ويب للمعلمين (React)
- قاعدة بيانات (Firebase)

## 🏗️ بنية المشروع

```
Halaqah/
├── mobile-app/          # تطبيق Flutter للطلاب
├── admin-dashboard/     # لوحة تحكم React للمعلمين
├── shared/             # ملفات مشتركة
└── README.md
```

## ✨ المميزات المبسطة

### 📱 للطلاب:
- واجهة عربية بسيطة
- خطة يومية واضحة
- تسجيل إنجاز بنقرة واحدة
- إشعارات تلقائية

### 💻 للمعلمين:
- جدول يومي موحد
- تعديل سريع للحالات
- تقارير بسيطة
- إدارة الطلاب السهلة

## 🚀 بدء الاستخدام

1. تثبيت المتطلبات
2. إعداد Firebase
3. تشغيل التطبيقات
